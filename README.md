## Como Iniciar o Projeto ?

1. Clone o repositorio:
  ```bash
     git clone https://github.com/Jv1ctor/TCC-mobile-app-chat
  ```

2. Abra no Android Studio


3. Adicione a url e key do supabase no **local.properties**:
   ```properties
      SUPABASE_KEY=[secret_key]
      SUPABASE_URL=[your_url_supabase]
   ```


5. Build o projeto

6. Run app
